# Social Media Automation UI

A modern React-based user interface for managing social media uploads across multiple platforms.

## Features

- Modern, responsive design with Chakra UI
- Drag-and-drop file upload
- Real-time upload progress tracking
- Platform-specific status monitoring
- Error handling and direct links to failed uploads

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- React with TypeScript
- Chakra UI for components
- React Dropzone for file uploads
- React Icons for platform icons
- Framer Motion for animations
